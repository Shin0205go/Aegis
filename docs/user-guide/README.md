# ユーザーガイド

AEGISを使用する方向けのガイドです。基本的な使い方から応用まで、段階的に学べるように構成されています。

## 📚 ガイド一覧

### 🚀 基本編

1. **[導入・初期設定](./getting-started.md)**
   - システム要件
   - インストール手順
   - 環境設定
   - 動作確認

2. **[Claude Desktop統合](./claude-desktop-setup.md)**
   - Claude Desktopとの連携設定
   - 設定ファイルの編集
   - トラブルシューティング

3. **[Web UI使用方法](./web-ui.md)**
   - ダッシュボードの見方
   - ポリシー管理画面
   - 監査ログの確認

### 📝 実践編

4. **[ポリシー記述ガイド](./policy-writing.md)**
   - 自然言語ポリシーの基本
   - 効果的なポリシーの書き方

5. **[利用可能なツール一覧](./tools-catalog.md)**
   - ファイルシステム操作ツール
   - コード実行ツール

6. **[実践的な使用例](./examples.md)**
   - 基本的な使用例
   - 応用的なシナリオ
   - ベストプラクティス

## 🎯 クイックスタート

初めての方は、以下の順序で進めることをお勧めします：

1. [導入・初期設定](./getting-started.md) でAEGISをインストール
2. [Claude Desktop統合](./claude-desktop-setup.md) で連携設定
3. [Web UI使用方法](./web-ui.md) で管理画面に慣れる
4. [ポリシー記述ガイド](./policy-writing.md) で基本的なポリシーを作成

## 💡 ヒント

- 各ガイドには実際の画面例やコマンド例が含まれています
- トラブルが発生した場合は、各ガイドの「トラブルシューティング」セクションを確認してください
- より詳細な技術情報が必要な場合は、[開発者ガイド](../developer-guide/) を参照してください
