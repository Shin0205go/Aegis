## バージョニングポリシー

AEGISはセマンティックバージョニング (SemVer) を採用しています：

- **メジャーバージョン (X.0.0)**: 後方互換性のない変更
- **マイナーバージョン (0.X.0)**: 後方互換性のある機能追加
- **パッチバージョン (0.0.X)**: 後方互換性のあるバグ修正

## サポートポリシー

| バージョン | ステータス | サポート終了日 |
|-----------|-----------|---------------|
| 1.0.x | **Active** | 2026-06-19 |
| 0.9.x | Maintenance | 2025-12-31 |
| 0.8.x | End of Life | 2025-09-30 |
| < 0.8 | End of Life | - |

- **Active**: 新機能、改善、バグ修正、セキュリティ更新
- **Maintenance**: 重要なバグ修正とセキュリティ更新のみ
- **End of Life**: サポート終了

## アップグレードガイド

### 0.9.x から 1.0.0 へ

1. **環境変数の更新**
   ```bash
   # 旧
   POLICY_ENGINE_TYPE=legacy
   
   # 新
   ENFORCEMENT_SYSTEM_ENABLED=true
   ```

2. **設定ファイルの移行**
   ```bash
   npm run migrate:config
   ```

3. **ポリシーファイルの確認**
   - 自動移行ツールを実行: `npm run migrate:policies`
   - 移行後のポリシーをテスト: `npm run test:policies`

4. **APIクライアントの更新**
   - SDK を最新版に更新
   - 非推奨APIの置き換え

### 0.8.x から 0.9.x へ

1. **新しい依存関係のインストール**
   ```bash
   npm install
   npm run build
   ```

2. **制約・義務の設定更新**
   - `constraints.json` を `enforcement-config.json` に移行
   - カスタムプロセッサの再実装が必要な場合あり
